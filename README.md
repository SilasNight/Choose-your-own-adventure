# Choose Your Own Adventure Game
This is a small small game that I made just to check if I could. It <br>
is a multiple choice adventure game.
___
### Installation 
You need to decide where you want this code to be downloaded to<br>
make a new folder and name it whatever you like<br>
after that make sure you have a version of python on your computer <br>
https://www.python.org/downloads/

make sure you have git installed on your computer as well<br>
https://git-scm.com/install/

Now navigate to you the folder you want to use with<br>
cd (your filepath)

Get the project<br>
git clone https://github.com/SilasNight/Choose-your-own-adventure

Create a virtual enviroment for the project<br>
python3 -m venv .venv

Navigate to the project<br>
cd Choose-your-own-adventure

Install the requirements<br>
pip install -r requirements.txt

Run the program<br>
py main.py
